How to Set Up Gmail for ArrayForms

Gmail App Password Setup Guide

Important: For security reasons, Gmail doesn't allow regular passwords for third-party apps. You need to create a special "App Password" to use your Gmail account with ArrayForms.
Prerequisites:
  • You must have 2-Step Verification enabled on your Google Account
  • If you haven't set up 2-Step Verification yet, follow Google's guide to set it up first
Step-by-Step Instructions:
  1. Go to your Google Account
  2. In the navigation panel, select Security
  3. Under "Signing in to Google," select 2-Step Verification
  4. At the bottom of the page, select App passwords
  5. Enter a name that helps you remember what this app password is for (e.g., "ArrayForms Email Sender")
  6. Select Create
  7. Google will display a 16-character password. Copy this password
  8. Return to ArrayForms and paste this password in the "Email Password/App Password" field

Example of Google's App Password generation screen

Important Security Notes:
  • App passwords are 16 characters long with no spaces
  • You can revoke access at any time from your Google Account
  • ArrayForms does not store your password anywhere
